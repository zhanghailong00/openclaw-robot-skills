## 一、项目概述

### 1.1 功能定义

家具整理 Skill 实现了桌面物品的**自动识别、分类排序、路径规划和抓取放置**。

用户只需说"整理桌子"，系统就会自动：
1. 摄像头拍照识别物品
2. 按优先级排序
3. 逐个抓取并放置到指定区域

### 1.2 核心价值

| 价值点 | 说明 |
|--------|------|
| **自动化** | 无需人工干预，全自动完成 |
| **智能化** | 自动识别、排序、规划路径 |
| **可扩展** | 支持新物品类型、自定义放置规则 |
| **可复用** | 复用现有 Skill，开发成本低 |

### 1.3 技术指标

| 指标 | 目标值 | 当前值 |
|------|--------|--------|
| 检测准确率 | >90% | ~70% |
| 抓取成功率 | >95% | ~90% |
| 单次抓放时间 | <10s | ~8s |
| 支持物品数 | 4+ | 4（可乐、汉堡、薯条、盘子） |

### 3.2 路径规划模块（02_plan_path.py）

#### 功能

- 根据起点、终点规划运动路径
- 支持简单避障
- 生成安全的运动轨迹

#### 技术实现

**1. 安全高度策略**

```python
SAFE_HEIGHT = 80  # 所有水平移动都在此高度

# 路径规划逻辑
def plan_path(start, end, obstacles):
    path = []

    # 1. 抬升到安全高度
    if start[2] < SAFE_HEIGHT:
        path.append([start[0], start[1], SAFE_HEIGHT])

    # 2. 水平移动（在安全高度）
    path.append([end[0], end[1], SAFE_HEIGHT])

    # 3. 下降到目标位置
    path.append([end[0], end[1], end[2]])

    return path
```

```python
def grab_object(grab_pos):
    """抓取物品流程"""
    # 1. 移动到抓取位置上方（安全高度）
    move_arm(grab_pos[0], grab_pos[1], SAFE_HEIGHT)

    # 2. 打开夹爪
    control_gripper(open=True)

    # 3. 下降到抓取位置
    move_arm(grab_pos[0], grab_pos[1], grab_pos[2])

    # 4. 关闭夹爪（抓取）
    control_gripper(open=False)

    # 5. 抬起到安全高度
    move_arm(grab_pos[0], grab_pos[1], SAFE_HEIGHT)
```

**4. 放置流程**

```python
def place_object(place_pos):
    """放置物品流程"""

    move_arm(place_pos[0], place_pos[1], SAFE_HEIGHT)

    # 2. 下降到放置位置
    move_arm(place_pos[0], place_pos[1], place_pos[2])

    # 3. 打开夹爪（放置）
    control_gripper(open=True)

    # 4. 抬起到安全高度
```

## 六、出现的问题与解决方案

### 6.1 问题1：视觉检测结果解析失败

**现象：**
```json
{"error": "视觉检测结果解析失败", "status": "error"}
```

**原因：**
vision-detect 脚本输出包含多行内容（NPU初始化日志 + JSON结果），直接 `json.loads()` 会失败。

**解决方案：**
```python
# 从stdout中提取JSON（查找第一个{和最后一个}）
stdout = result.stdout.strip()
start = stdout.find('{')
end = stdout.rfind('}') + 1
json_str = stdout[start:end]
data = json.loads(json_str)
```

---

### 6.2 问题2：子脚本调用返回多个JSON对象

**现象：**
```
[DEBUG] 找到 2 个JSON对象
检测失败: 脚本执行失败
```

**原因：**
01_detect_and_sort.py 输出了两个JSON：
1. vision-detect 的输出（detections）
2. 自己的输出（objects）

`json.loads()` 遇到第二个 `{` 时报错。

**解决方案：**
```python
# 查找所有JSON对象，返回最后一个
json_objects = []
i = 0
while i < len(stdout):
    start = stdout.find('{', i)
    # ... 解析逻辑 ...
    json_objects.append(obj)

return json_objects[-1]  # 返回最终结果
```

### 6.4 问题4：机械臂移动参数格式错误

### 6.7 问题7：机械臂位置偏差

**现象：**
```
target: {"x": 100.0, "y": 0.0, "z": 80.0}
actual: {"x": 167.6, "y": 49.6, "z": 66.2}
```

**原因：**
目标位置超出机械臂工作范围，或者逆运动学求解有误差。

**解决方案：**
- 当前版本：可接受（不影响抓取）
- 优化版本：添加工作范围检查，自动调整目标位置
