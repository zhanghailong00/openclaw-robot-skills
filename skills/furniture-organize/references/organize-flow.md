# 家具整理流程说明

## 概述

家具整理技能实现了桌面物品的自动识别、优先级排序、路径规划和抓取放置。

## 整体流程

│                    家具整理流程                              │

## 详细步骤

### 步骤1：视觉检测

**输入：** 摄像头图像
**输出：** 物品列表（像素坐标）

调用 `vision-detect` 的 `02_run_detection.py`：
```bash
python 02_run_detection.py
```

返回格式：
```json
{
  "detections": [
    {"class": "cola", "cx": 240, "cy": 185, "confidence": 0.95},
    {"class": "hanbao", "cx": 320, "cy": 200, "confidence": 0.92}
  ]
}
```

### 步骤2：坐标转换

**输入：** 像素坐标 (cx, cy)
**输出：** 机械臂坐标 (arm_x, arm_y, arm_z)

转换过程：
1. 像素坐标 → 工作台坐标（通过相机内参和手眼标定）
2. 工作台坐标 → 机械臂坐标（通过 `tf_ws2arm()`）

调用 `coord-transform` 的 `03_pixel2ws.py` 和 `04_ws2arm.py`

### 步骤3：优先级排序

**输入：** 物品列表
**输出：** 排序后的物品列表

排序规则：
| 优先级 | 物品 | 原因 |
|--------|------|------|
| 3（最高） | 可乐 | 饮料先放，防止倾倒 |
| 2 | 汉堡 | 主食其次 |
| 1（最低） | 薯条 | 配餐最后 |

同类型物品按距离中心远近排序，近的先抓。

### 步骤4：路径规划

**输入：** 起点、终点、障碍物
**输出：** 运动路径点列表

路径规划策略：
```
1. 当前位置 → 安全高度（80mm）
2. 检查障碍物
   - 有障碍物：绕行到障碍物侧面
   - 无障碍物：直接水平移动
3. 移动到目标上方
4. 下降到目标位置
```

安全高度：80mm（所有水平移动都在此高度进行）

### 步骤5：抓取执行

**输入：** 抓取位置
**输出：** 抓取结果

执行步骤：
```
1. 移动到抓取位置上方 (x, y, 80)
2. 打开夹爪 (25°)
3. 下降到抓取位置 (x, y, z)
4. 关闭夹爪 (0°) - 抓取
5. 抬起到安全高度 (x, y, 80)
```

### 步骤6：放置执行

**输入：** 放置位置
**输出：** 放置结果

执行步骤：
```
1. 移动到放置位置上方 (x, y, 80)
2. 下降到放置位置 (x, y, z)
3. 打开夹爪 (25°) - 放置
4. 抬起到安全高度 (x, y, 80)
```

## 默认配置

### 放置区域

| 物品 | 放置位置（机械臂坐标） | 说明 |
|------|------------------------|------|
| 可乐 | (100, 50, -31.2) | 饮料区 |
| 汉堡 | (100, 0, -31.2) | 主食区 |
| 薯条 | (100, -50, -31.2) | 配餐区 |

### 机械臂参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 安全高度 | 80mm | 所有水平移动高度 |
| 抓取高度 | -31.2mm | 物品底部高度 |
| 夹爪打开 | 25° | 抓取/放置时打开角度 |
| 夹爪关闭 | 0° | 抓取时关闭角度 |

## 避障策略

### 简单避障（当前实现）

1. **检测碰撞：** 检查路径线段是否与障碍物球体相交
2. **计算绕行点：** 在障碍物侧面选择绕行点
3. **绕行方向：** 根据障碍物相对路径的位置决定左侧或右侧绕行

### 进阶避障（可选）

- A* 算法：网格化地图，寻找最短路径
- RRT 算法：随机采样，适合复杂环境

## 常见问题

### Q1: 检测不到物品

**可能原因：**
- 摄像头未连接或权限不足
- 物品不在视野范围内
- YOLO 模型未加载

**解决方法：**
```bash
# 测试摄像头
python /path/to/vision-detect/scripts/01_check_camera.py
```

### Q2: 坐标转换错误

**可能原因：**
- 相机标定参数过期
- 手眼标定不准确

**解决方法：**
```bash
# 重新标定
python /path/to/coord-transform/scripts/01_calibration.py
python /path/to/coord-transform/scripts/02_hand_eye.py
```

### Q3: 抓取失败

**可能原因：**
- 抓取位置不准确
- 夹爪力度不够
- 物品滑动

**解决方法：**
- 检查坐标转换是否正确
- 调整夹爪参数
- 降低抓取速度

### Q4: 路径规划绕行不当

**可能原因：**
- 障碍物信息不准确
- 绕行策略过于简单

**解决方法：**
- 手动指定障碍物位置
- 调整绕行距离参数
