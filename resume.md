# 简历项目描述

## 项目名称

基于 AI 智能体的嵌入式机器人自然语言控制系统

## 项目背景

为教育实训箱设计 AI 智能控制系统，使学生能通过自然语言（飞书/微信）控制机械臂和传送带等硬件设备，降低机器人编程学习门槛。

## 技术栈

| 类别 | 技术 |
|------|------|
| 硬件平台 | RK3588 ARM64 开发板（OrangePi AIpro 20T） |
| 机器人 | 4-DOF 机械臂（UART 串口舵机）、传送带（GPIO）、UVC 摄像头 |
| AI 框架 | OpenClaw 开源 AI 智能体框架 |
| 大语言模型 | DeepSeek（云端 API） |
| 视觉检测 | YOLOv11 + Ascend 310B NPU 推理加速 |
| 通信协议 | UART 串口、GPIO、I2C、WebSocket |
| 开发语言 | Python、Node.js |
| 平台工具 | Linux (Ubuntu)、systemd、飞书开放平台 |

## 主要工作

### 1. 嵌入式平台 AI 智能体部署

- 在 RK3588 ARM64 开发板上部署 OpenClaw AI 智能体框架（Node.js 服务）
- 解决 ARM64 平台兼容性问题：Node.js 内存管理优化（V8 heap 调优）、npm 包交叉编译
- 配置 systemd 服务实现开机自启，解决进程守护和日志管理
- 接入飞书开放平台，通过 WebSocket 长连接实现自然语言交互

### 2. 硬件控制 Skill 插件架构设计

- 设计并实现 5 个独立可复用的 Skill 模块：
  - **arm-basic**：机械臂基础控制（状态查询、归零、移动、夹爪）
  - **vision-detect**：YOLO 目标检测（拍照、去畸变、NPU 推理）
  - **coord-transform**：坐标转换（像素→工作台→机械臂，含 Z 轴误差校正）
  - **conveyor-belt**：传送带控制（GPIO 启停、红外/超声波传感器）
  - **arm-delivery**：送餐流程编排（组合以上 Skill 完成完整任务）
- 编写 SKILL.md 描述文件，定义工具输入输出格式，使 AI 智能体能自主理解任务意图并编排调用
- 设计渐进式披露架构：SKILL.md 放概览，references/ 放细节，优化 AI 上下文利用率

### 3. 机器人运动学与视觉引导

- 实现 4-DOF 机械臂正逆运动学（DH 参数法）和最小抖动轨迹规划
- 完成相机-机械臂手眼标定（九点法），实现像素坐标到机械臂坐标的精确转换
- 集成 YOLOv11 目标检测模型，在 Ascend 310B NPU 上实现实时推理（3-5 秒/帧）
- 实现完整视觉引导抓取链路：YOLO 检测 → 像素坐标转换 → 逆运动学解算 → 舵机控制

### 4. 系统集成与调试

- 排查并解决多个集成问题：ARM64 Node.js OOM、GPIO 逻辑反转、串口通信超时、坐标变换精度
- 实现传感器驱动（I2C 红外/超声波）与传送带的联动控制
- 设计安全保护机制：关节限位自动钳位、串口通信异常处理、GPIO 急停

## 技术亮点

| 方向 | 内容 |
|------|------|
| AI + 硬件 | 大语言模型 + 工具调用（Function Calling）模式控制物理硬件 |
| 嵌入式 AI | ARM64 + NPU 推理加速，边缘端实时视觉检测 |
| 机器人学 | DH 参数运动学、最小抖动轨迹规划、九点手眼标定 |
| 系统架构 | Skill 插件化设计，AI 自主编排硬件操作 |
| 工程实践 | 跨平台部署、串口通信、GPIO/I2C 驱动、进程守护 |

## 项目成果

- 实现通过飞书消息自然语言控制机械臂完成抓取、放置、传送带运输等操作
- AI 智能体能自主理解任务意图，自动编排多个硬件模块完成复杂任务
- 系统响应时间 < 6 秒（含 AI 推理 + 硬件执行）
- Skill 插件架构支持快速扩展新硬件模块

## 面试话术

**项目一句话介绍：**
> 在 RK3588 嵌入式平台上部署 AI 智能体，让学生通过飞书发消息就能控制机械臂，AI 自己理解任务、自己调用硬件接口。

**遇到的难点：**
> 1. ARM64 平台上 Node.js 内存管理问题，OpenClaw 命令卡死，需要绕过包装脚本直接调用
> 2. AI 智能体输出的坐标是像素坐标，需要经过仿射变换、齐次变换、Z 轴误差校正三步才能给机械臂用
> 3. 串口舵机的通信协议是自定义的，需要理解二进制包格式才能对接

**收获：**
> 理解了 AI 智能体的架构设计（Skill 插件 + 工具调用），掌握了嵌入式 Linux 上部署 AI 服务的完整流程，学会了从硬件驱动到上层应用的全栈开发。
