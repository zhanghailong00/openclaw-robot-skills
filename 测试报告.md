## 二、测试类型

本次测试涵盖以下测试类型：

| 测试类型 | 英文名称 | 说明 |
|---------|---------|------|
| 功能测试 | Functional Testing | 验证每个 Skill 能否正常工作 |
| 集成测试 | Integration Testing | 验证多个 Skill 能否协同工作 |
| 系统测试 | System Testing | 验证整体流程是否正确 |
| 验收测试 | User Acceptance Testing (UAT) | 验证是否满足用户需求 |
| NLP 测试 | NLP Testing | 验证 AI 理解自然语言的能力 |
| 异常测试 | Exception Testing | 验证错误处理能力 |
| 端到端测试 | End-to-End (E2E) Testing | 从用户指令到硬件执行的完整链路 |
| 场景测试 | Scenario-based Testing | 基于真实使用场景的测试 |
| 探索性测试 | Exploratory Testing | 测试意外输入的处理能力 |
